This plugin is an chart add-on for the Elementor Page Builder

== Description ==
This plugin is an add-on for the Elementor Page Builder, providing a simple and clean solution for integrating charts. It allows you to effortlessly include animated, interactive graphs on your website. You can easily customize the style using this widget.

For more details :[Elementor Chart](https://wpocean.com/wp/plugins/chart).
For our other elementor & WP works visit : [wpOcean](https://wpocean.com). 


== Installation ==
Installation is fairly straight forward. Install it from the WordPress plugin repository.
